# Mathematical Machine Learning with Python

A collection of implementations, experiments, and visualizations developed while studying Mathematical Machine Learning.

The goal of this repository is to strengthen the mathematical foundations behind modern machine learning algorithms through hands-on implementation in Python.

---

## Topics Covered

### Machine Learning Foundations

* Linear Regression
* Logistic Regression
* Polynomial Regression
* Clustering & Graph Clustering
* Best-Fit Subspaces

### Machine Learning Theory

* Singular Value Decomposition
* Curse and Blessing of High Dimensionality
